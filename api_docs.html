Nigeria Economic Data API

Free Public REST API - No authentication required - No rate limits

11,022 observations across 115 indicators — GDP growth, inflation, exchange rates, MPR, FX reserves, multi-currency CBN rates, GDP sectors, CBN balance sheet and NFEM daily data.

Base URL: https://web-production-feb5f.up.railway.app

API Endpoint Reference

All endpoints use HTTP GET and return JSON. CORS is fully open. No keys required.

GET/api/summary

Latest single observation for all five core indicators in one call. Ideal for dashboard KPI cards.

{"gdp_growth":{"obs_date":"2024-09-30","value":3.97},"inflation":{"obs_date":"2024-12-01","value":34.8},"exchange_rate":{"obs_date":"2024-12-01","value":1535.0},"mpr":{"obs_date":"2024-11-26","value":27.5},"fx_reserves":{"obs_date":"2026-03-01","value":49.7}}
Try it
GET/api/gdp

Quarterly real GDP growth rate (%) and annual nominal GDP in USD billions.

Optional: start (default 2020-01-01) end (default 2026-12-31)

{"gdp_growth":[{"obs_date":"2020-01-01","value":1.87},{"obs_date":"2020-04-01","value":-6.10},...], "gdp_usd":[{"obs_date":"2020-01-01","value":432.29},...], "unit_growth":"Percent (%)","unit_usd":"USD Billions","source":"NBS (growth), World Bank (USD)"}
Try it
GET/api/inflation

Monthly headline inflation rate (year-on-year CPI change). Source: NBS Consumer Price Index.

Optional: start end

{"data":[{"obs_date":"2020-01-01","value":12.13},{"obs_date":"2020-02-01","value":12.2},...], "unit":"Percent (%)","source":"NBS CPI Report"}
Try it
GET/api/exchange-rate

Monthly average NGN/USD official central rate. Captures the full impact of the June 2023 FX unification reform.

Optional: start end

{"data":[{"obs_date":"2020-01-01","value":360.0},{"obs_date":"2023-07-01","value":782.54},{"obs_date":"2024-01-01","value":1497.0},...], "unit":"Naira per USD","source":"CBN Official Rate"}
Try it
GET/api/interest-rate

CBN Monetary Policy Rate (MPR) at each MPC decision date — stepped series from 11.5% to 27.5% between 2020 and 2024.

Optional: start end

{"data":[{"obs_date":"2020-01-21","value":13.5},{"obs_date":"2020-03-24","value":13.5},{"obs_date":"2022-05-24","value":13.0},...,{"obs_date":"2024-11-26","value":27.5}]}
Try it
GET/api/fx-reserves

Monthly gross, liquid and blocked foreign exchange reserves in USD billions, plus block percentage. CBN data 2020-2026.

{"gross":[{"obs_date":"2020-01-01","value":38.59},...], "liquid":[{"obs_date":"2020-01-01","value":36.4},...], "blocked":[{"obs_date":"2020-01-01","value":2.19},...], "block_pct":[{"obs_date":"2020-01-01","value":5.68},...] }
Try it
GET/api/currency-circulation

Currency in circulation (NGN billions) — full history from 2002 to 2024. Source: CBN Annual Report.

{"data":[{"obs_date":"2002-12-01","value":368.4},{"obs_date":"2003-12-01","value":424.8},...,{"obs_date":"2024-06-01","value":4120.3}]}
Try it

NFEM (Nigeria Foreign Exchange Market) daily rates — closing, highest, lowest and weighted average. December 2024 to April 2026.

{"closing":[{"obs_date":"2024-12-02","value":1535.0},...], "highest":[...], "lowest":[...], "weighted_avg":[{"obs_date":"2024-12-02","value":1534.2},...] }
Try it
GET/api/multicurrency

Buying, central and selling rates for 12 currencies (USD, GBP, EUR, CNY, JPY, CHF, ZAR, AED, SAR, SDR, CFA, WAUA). Monthly CBN data 2020-2026.

{"usd":{"buying":[{"obs_date":"2020-01-01","value":357.5},...], "central":[{"obs_date":"2020-01-01","value":360.0},...], "selling":[{"obs_date":"2020-01-01","value":362.5},...] },"gbp":{...},"eur":{...},...}
Try it
GET/api/gdp-sectors

Real GDP by sector — agriculture, industry, services, manufacturing and 8 more sectors. NBS data from 1981 to 2024 (quarterly and annual). Values in NGN billions at constant 2010 prices.

{"agriculture":[{"obs_date":"1981-01-01","value":14820.5},...,{"obs_date":"2024-01-01","value":22541.3}], "industry":[...], "services":[...], "manufacturing":[...], ...}

CBN assets and liabilities — total assets, total liabilities, gold holdings, currency issued, government deposits and bankers deposits. Monthly data 2005-2023.

{"total_assets":[{"obs_date":"2005-01-01","value":3211.4},...], "total_liabilities":[...], "gold":[...], "currency_issued":[...], "govt_deposits":[...], "bankers_deposits":[...] }
Try it

Use Cases

For Researchers

Download complete time series for econometric modelling, regression analysis, and policy papers. All data is deduplicated, date-normalised to ISO 8601, and sourced directly from CBN, NBS and World Bank publications.

For Developers

Integrate Nigerian economic data into apps, dashboards or data pipelines with zero setup. CORS is fully open, all endpoints return clean JSON arrays, and there is no authentication overhead.

For Students

Use the data for final year projects, dissertations and academic analysis in economics, data science, finance or computer science. Free, open and citable.

For Journalists

Verify economic claims instantly using the /api/summary endpoint for the latest snapshot. The full time series endpoints provide context for historical comparisons and fact-checking.
