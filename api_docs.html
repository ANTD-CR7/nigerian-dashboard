NPEDATA / API Docs

Nigeria Economic Data — Open API

Free. No authentication. No rate limits.

https://web-production-feb5f.up.railway.app
Note on Data Source

All data is fetched directly from Supabase. The Supabase REST API is also publicly accessible at https://fjsytcmcxapfbrwvawmu.supabase.co/rest/v1/observations using the anon key shown in all page source files.

Endpoints

GET
/api/summary

Latest single value for all five core indicators — GDP, Inflation, Exchange Rate, MPR, FX Reserves. Ideal for KPI dashboards.

Try it →
GET
/api/gdp

Quarterly real GDP growth rate (%) and annual nominal GDP in USD billions. Optional: start and end date params.

Try it →
GET
/api/inflation

Monthly headline inflation rate (year-on-year CPI change). Optional date range params. Source: NBS.

Try it →
GET
/api/exchange-rate

Monthly average NGN/USD official central rate. Captures the full impact of the June 2023 FX unification reform.

Try it →
GET
/api/interest-rate

CBN Monetary Policy Rate (MPR) at each MPC decision date. Stepped series from 11.5% to 27.5% 2020–2024.

Try it →
GET
/api/fx-reserves

Monthly gross, liquid and blocked foreign exchange reserves plus block percentage. CBN data 2020–2026.

Try it →
GET
/api/currency-circulation

Currency in circulation (NGN millions) — full history from 2002 to 2024. Source: CBN Annual Report.

Try it →
GET
/api/nfem

NFEM daily rates — closing, highest, lowest and weighted average. December 2024 to April 2026.

Try it →
GET
/api/multicurrency

Buying, central and selling rates for 12 currencies (USD, GBP, EUR, CNY, JPY, CHF, ZAR, AED, SAR, SDR, CFA, WAUA).

Try it →
GET
/api/gdp-sectors

Real GDP by sector — agriculture, industry, services, manufacturing and 8 more. NBS data 1981–2024.

Try it →
GET
/api/cbn-balance-sheet

CBN balance sheet — total assets, liabilities, gold, currency issued, govt deposits, bankers deposits. 2005–2023.

Try it →
GET
/api/analytics

Combined dataset for inflation, exchange rate, FX reserves, GDP and MPR in a single call. Optimised for analysis.

Try it →

Use Cases

For Researchers

Download complete time series for econometric modelling. All data is deduplicated, date-normalised to ISO 8601, and sourced from primary CBN, NBS and World Bank publications.

For Developers

Integrate Nigerian economic data into your own applications with zero authentication. CORS is fully open and all responses are standard JSON arrays.

For Students

Use the data for dissertations, final year projects and academic analysis in economics, data science, finance or computer science. Free and openly accessible.
