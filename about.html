About This System

Design and Implementation of a Web-Based Financial Data Aggregation and Analytics System Using Nigerian Public Economic Data

Mission

"This system aggregates 11,022 observations across 115 indicators from the CBN, NBS and World Bank into a single interactive web dashboard — making Nigeria's public economic data accessible to researchers, developers, students and the general public."

System Overview

This system aggregates four core Nigerian macroeconomic indicators from three official public data sources for the period January 2020 to December 2026. It presents the data through an interactive web dashboard with line charts and summary tables that any user can access through a standard web browser.

This project was developed as a Final Year Project at Caleb University, Imota, Lagos, in the Department of Computer Science. Its contribution is a free, open-source, and academically documented implementation built on a teachable technology stack. The database now contains 11,022 observations across 115 indicators, covering GDP sectoral breakdown (57 sectors, 1981–2024), multi-currency CBN rates (12 currencies), FX reserves, CBN balance sheet, NFEM daily rates and the original four core macro indicators.
