NPEDATA / Roadmap

Roadmap

Where the platform is, where it's going, and how you can shape it

The idea

NPEDATA is an open, free platform for Nigeria's public economic data. It's built in the open, and it grows in the open. This page is the honest state of things — what already works, what's being built next, and what's still just an idea. If something you need is missing, tell us.

✅ Live now

  • 122 indicators, 12,100 records — CBN, NBS and World Bank, 2020–2026 (some series back to 1960).
  • Free open REST API — no key, fully documented, HATEOAS Level 3.
  • Forecasting — Holt-Winters with 95% confidence bands.
  • Lead/Lag Explorer & Seasonal Decomposition — see which indicators move first, and split any series into trend and season.
  • Integrity features — spurious-correlation warnings, significance testing, honest charts.
  • Developer SDKs — official Python and JavaScript clients.
  • AI "ask the data" — plain-language questions grounded in real figures (opt-in).

🚧 Next up

  • Automated data collection — World Bank is already automated; CBN/NBS connectors are next so figures refresh on their own.
  • Forecasts on every indicator page — bring the forecast out of Analytics and onto the individual charts.
  • More coverage — additional indicators and longer historical series where the sources allow.
  • Data-freshness indicators — show clearly how recent each series is.

🔭 Later / exploring

  • State-level data — beyond national aggregates.
  • API tiers & higher limits — for heavier programmatic consumers.
  • Always-on infrastructure — removing cold starts as traffic grows.
  • Community-requested datasets — driven by what people actually ask for below.

💬 Help shape it

This platform gets better when people say what they need. Everything runs through the public issue tracker on GitHub — no account details required beyond a free GitHub login.

📊 Request a dataset 🐛 Report something wrong 💡 Suggest a feature
