# Nigerian Financial Data Aggregation System
## Step-by-Step Setup Instructions

---

### WHAT YOU NEED ON YOUR LAPTOP
- Python 3.10 or newer
- Internet connection

---

### STEP 1 — Create the database tables in Supabase

1. Go to supabase.com and open your project
2. Click **SQL Editor** in the left menu
3. Open the file `database/setup.sql`
4. Copy everything and paste it into the SQL Editor
5. Click **Run**
6. You should see "Success" — your tables are created

---

### STEP 2 — Put your data files in the project folder

Copy these files into the SAME folder as `load_data.py`:
- `Inflation_Data_in_Excel.xlsx`
- `exchange_rate.xlsx`
- `mpr.xlsx`

---

### STEP 3 — Install Python packages

Open your terminal (Command Prompt on Windows, Terminal on Mac) and run:

```
pip install -r requirements.txt
```

---

### STEP 4 — Load all data into Supabase

In the same terminal, run:

```
python load_data.py
```

You will see each value being inserted. Wait for it to finish.
It should say: **ALL DATA LOADED SUCCESSFULLY**

---

### STEP 5 — Start the API server

In the terminal, run:

```
uvicorn main:app --reload
```

You will see: **Uvicorn running on http://127.0.0.1:8000**

Leave this terminal window open. Do not close it.

---

### STEP 6 — Open the dashboard

Open your web browser and go to:

```
http://localhost:8000/app/index.html
```

You should see the Nigeria Economic Dashboard with all four charts and data.

---

### BONUS — View the API documentation

Go to:
```
http://localhost:8000/docs
```

This shows all your API endpoints automatically generated by FastAPI.

---

### FOLDER STRUCTURE

```

    └── css/

```

---

### IF SOMETHING GOES WRONG

**"Could not reach API"** on the browser:
- Make sure you ran `uvicorn main:app --reload` and it is still running

**Error during `python load_data.py`**:
- Check the Excel files are in the same folder as `load_data.py`
- Check you ran setup.sql in Supabase first

**Tables already exist error in Supabase**:
- That is fine — the SQL uses `IF NOT EXISTS` so it is safe to run again
