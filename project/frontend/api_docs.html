Nigeria Economic Data API

Free Public REST API - No authentication - No rate limits

Programmatic access to Nigerian macroeconomic data including GDP, inflation, exchange rates, monetary policy, foreign reserves and multi-currency CBN rates - 2020 to 2026.

Base URL: https://web-production-feb5f.up.railway.app

Endpoints

All endpoints return JSON. GET requests only. CORS enabled for all origins.

GET /api/summary

Latest single value for all core indicators: GDP growth, inflation, exchange rate, MPR, and FX reserves. Ideal for KPI dashboards.

{"gdp_growth":{"obs_date":"2024-09-30","value":3.97},"inflation":{"obs_date":"2024-12-01","value":34.8},"exchange_rate":{"obs_date":"2024-12-01","value":1535.0},"mpr":{"obs_date":"2024-11-26","value":27.5},"fx_reserves":{"obs_date":"2026-03-01","value":49.7}}
Try it
GET /api/gdp

Quarterly real GDP growth rate (%) and annual nominal GDP in USD billions.

Query params: start (default 2020-01-01)   end (default 2026-12-31)

{"gdp_growth":[{"obs_date":"2020-01-01","value":1.87},...], "gdp_usd":[{"obs_date":"2020-01-01","value":432.29},...], "unit_growth":"Percent (%)","unit_usd":"USD Billions"}
Try it
GET /api/inflation

Monthly headline inflation rate (%) — year-on-year CPI change. Source: NBS Consumer Price Index Report.

Query params: start   end

{"data":[{"obs_date":"2020-01-01","value":12.13},{"obs_date":"2020-02-01","value":12.2},...], "unit":"Percent (%)","source":"NBS CPI Report"}
Try it
GET /api/exchange-rate

Monthly average NGN/USD exchange rate (official CBN rate). Captures the full impact of the June 2023 FX unification reform.

Query params: start   end

{"data":[{"obs_date":"2020-01-01","value":360.0},...,{"obs_date":"2023-07-01","value":782.54},...], "unit":"Naira per USD"}
Try it
GET /api/interest-rate

CBN Monetary Policy Rate (MPR) at each MPC decision date. Stepped time series reflecting policy decisions from 2020 to 2026.

Query params: start   end

{"data":[{"obs_date":"2020-01-21","value":13.5},{"obs_date":"2020-03-24","value":13.5},...,{"obs_date":"2024-11-26","value":27.5}]}
Try it
GET /api/fx-reserves

Monthly gross foreign exchange reserves in USD billions. Covers 2020 to 2026 including the post-reform recovery to $49.7B by March 2026.

{"data":[{"obs_date":"2020-01-01","value":38.59},...,{"obs_date":"2026-03-01","value":49.7}]}
Try it
GET /api/currency-circulation

Currency in circulation data (NGN billions). Reflects CBN monetary supply management over time.

{"data":[{"obs_date":"2020-01-01","value":2120.5},...}]}
Try it
GET /api/all-currencies

Buying, central and selling rates for 12 currencies (USD, GBP, EUR, CNY, JPY, CHF, ZAR, AED, SAR, SDR, CFA, WAUA) from 2020 to 2026. Returns ~2,700 time-series points.

{"usd":{"buying":[{"obs_date":"2020-01-02","value":358.0},...], "central":[...], "selling":[...]},"gbp":{...},...}
Try it
GET /api/analytics

Combined dataset for all five core indicators in a single request. Optimised for cross-indicator analysis, correlation studies and multi-series charts.

{"inflation":[...],"exchange_rate":[...],"fx_reserves":[...],"gdp":[...],"mpr":[...]}
Try it

Use Cases

Researchers

Download complete time series for econometric modelling. All data is deduplicated, date-normalised to ISO 8601, and sourced from primary CBN, NBS and World Bank publications.

Developers

Integrate Nigerian economic data into your own applications with zero authentication overhead. CORS is fully open and all responses are standard JSON arrays.

Students

Use the data for final year projects, dissertations and class assignments in economics, data science, finance or computer science. Free and open access.

Journalists

Verify economic claims with primary data. The /api/summary endpoint gives the latest snapshot in under 200ms - bookmark it for rapid fact-checking.

Interactive Swagger Docs

Full Swagger UI with live request testing is available at the /docs endpoint. Includes all parameters, response schemas and the ability to run requests directly in the browser.
